# 安徽省普高专升本考试招生院校信息平台

一个为安徽省专科毕业生提供专升本考试招生信息的网站平台。

## 项目概述

本平台汇集了安徽省所有普通高校专升本考试的招生院校信息，为考生提供一站式查询服务。平台具有以下特点：

- **全面覆盖**：涵盖安徽省15+所专升本招生院校信息
- **智能搜索**：支持按院校、城市、专业等多维度搜索
- **详细页面**：每个院校都有独立的详情页面
- **响应式设计**：适配各种设备屏幕
- **美观UI**：现代化设计，良好用户体验

## 功能特性

### 首页功能
1. **院校统计**：显示招生院校总数、公办/民办院校数量
2. **智能搜索**：实时搜索院校信息
3. **筛选过滤**：按院校类型（公办/民办）、所在城市筛选
4. **院校卡片**：可点击的院校信息卡片，跳转到详情页
5. **响应式布局**：自适应不同屏幕尺寸

### 院校详情页功能
1. **基本信息**：院校概况、在校人数、专业数量等
2. **招生信息**：招生计划、专业设置、报考条件
3. **学校特色**：学校优势和特色介绍
4. **地址交通**：校区位置和交通指南
5. **返回导航**：方便返回首页

### 用户体验
- 流畅的动画效果和过渡
- 清晰的信息层次结构
- 易于操作的交互设计
- 现代化的UI界面

## 技术栈

- **前端**：HTML5, CSS3, JavaScript (原生)
- **样式**：Flexbox, Grid布局，CSS渐变，阴影效果
- **图标**：Font Awesome 6.0
- **字体**：系统字体，支持中文显示
- **响应式**：Media Queries，移动优先设计

## 使用方法

1. 直接打开 `index.html` 即可访问网站
2. 在首页使用搜索框查找院校
3. 点击筛选按钮按类型或城市筛选
4. 点击院校卡片查看详细信息
5. 详情页中使用返回按钮返回首页

## 开发说明

### 数据管理
院校数据存储在 `index.html` 中的JavaScript数组，便于维护和更新。

### 样式设计
使用CSS变量统一管理颜色主题，方便后续主题切换。

### 响应式设计
采用移动优先的设计原则，确保在小屏幕设备上的良好体验。
